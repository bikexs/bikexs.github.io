参考资料：
```java
书籍：

Java Concurrency in Practice，作者：Brian Goetz, Tim Peierls, Joshua Bloch, Joseph Bowbeer, David Holmes, Doug Lea。这本书是Java并发编程的经典之作，详细介绍了Java并发编程的基础知识、同步工具和高级主题。强烈推荐阅读。
https://jcip.net/

Concurrency: State Models and Java Programs，作者：Jeff Magee, Jeff Kramer。这本书主要关注状态模型和并发编程在Java中的实现，内容丰富且深入浅出。

The Art of Multiprocessor Programming，作者：Maurice Herlihy, Nir Shavit。这本书详细介绍了多处理器编程的基本概念、同步算法和并发数据结构。虽然书中的示例是用Java编写的，但其主题适用于任何编程语言。

在线资源：

Java Tutorials: Concurrency：这是Oracle官方的Java教程，其中有一节专门讲解Java并发编程。内容涵盖了线程的创建、同步、线程池、阻塞队列等主题，是学习Java并发编程的好起点。

Doug Lea's Concurrency JSR-166 Interest Site：Doug Lea是Java并发包（java.util.concurrent）的主要设计者之一。这个网站包含了关于Java并发的讨论、文章和示例代码，对于深入了解并发编程非常有帮助。

IBM developerWorks: Java concurrency series：IBM developerWorks网站上的这个系列文章详细讲解了Java并发编程的各个方面，从基本概念到高级主题，内容丰富且易于理解。
```
